Prework Study Guide

HTML

  • The head element contains information about a webpage that is not visible in the browser, such as meta data
  • The body element represents the visible content shown to the user
  • Although commonly used interchangably, a tag only one part of an element, whereas an element is both the openening and closing statement
  • HTML is the structural part of a website

CSS

  • CSS is used to provide style to a website
  • CSS stands for Cascading Style Sheets
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • CSS rules can be assigned to individual elements of a website
  • Alternatively a class attribute such as .card can be used a a selector to assign CSS rules to multiple elements.

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Javascript allows a user to interact with a website such as filling in forms or using a search function
  • A variable is a name container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script

    • Arrays

  • An array is a single variable that is used to hold a group of data
  • Arrays are typically used to hold data that is related
  • An array can store different types of data within the same array
  • The indexing of an array always starts with 0 and goes up incrementally by one
  • Different data types are declared in arrays in specific ways, strings are declared in quotation marks, numbers and boolean values are declared without any quotation marks

    • Conditional statements and Loops

  • Loops are a more efficient way of implementing repetitive code
  • A for loop has three sections in its declaration First the starting point, then the number of iterations, followed by how the initial variable is to changed, usually incremented by 1 after each iteration
  • Javascript is an Object Oriented Programming (OOP) language

    • Functions

  • A function is a set of instructions that tells the computer how to perform a certain task
  • A function groups a set of instructions such that it can be called anytime you want a particular task to be performed
  • To use functions, there are two steps, first the fuction needs to be defined and then it needs to be called.

  • Remember, it is always good practice to indent code that nests within arrays, loops or functions to give your work a good and appealing structure
