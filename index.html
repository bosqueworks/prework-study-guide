Prework Study Guide

HTML

  • The head element contains information about a webpage that is not visible in the browser, such as meta data
  • The body element represents the visible content shown to the user
  • Although commonly used interchangably, a tag only one part of an element, whereas an element is both the openening and closing statement
  • HTML is the structural part of a website

CSS

  • CSS is used to provide style to a website
  • CSS stands for Cascading Style Sheets

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Javascript allows a user to interact with a website such as filling in forms or using a search function
